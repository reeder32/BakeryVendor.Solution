## Technologies Used
- c#
- .Net
- Microsoft.NET.Test


# Get Started

- Open your favorite Terminal
## Installation
```sh
git clone https://github.com/reeder32/BakeryVendor.Solution.git
```
#### Run Tests
```sh
cd BakeryVendor.Tests
dotnet test
```
#### Run Program
```sh
cd BakeryVendor
dotnet run
```
[Open your browser here](http://localhost:5000)
